# Few-Shot Prompt Architect

A precision engineering tool designed to transform basic user tasks into structured, high-fidelity LLM instructions using domain-specific few-shot patterns.

## Overview
Prompt variance is a major challenge in LLM orchestration. This tool solves that by programmatically injecting **Few-Shot Examples** and **System Personas** into the prompt construction process, ensuring consistent and expert-level outputs.

## Key Features
- **Few-Shot Patterning:** Uses validated 'Input/Output' examples to guide model behavior.
- **Dynamic Personas:** Context-aware roles for Technical, Creative, and Marketing domains.
- **Utility-First UI:** Built with Streamlit for rapid, clean interaction.

## Technical Resolution
During development, I addressed specific challenges regarding **Ngrok tunnel synchronization** and **multi-port microservice routing** to allow this tool to run alongside other NLP services.
